# electron-vue-template

Electron + Vue + Vuetify template

## Project setup
1. Install vue cli 3

2. Run the following scripts
```
vue create sellit
cd sellit
vue add vuetify
vue add electron-builder
```

## Build project
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
